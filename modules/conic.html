圆锥曲线 · 急救包

最核心的公式 + 真题示范 + 踩坑提醒。大题做法去 大题模板页。

1公式速查

公式过长时可左右滑动 ↔

椭圆 · 标准方程（焦点在 x 轴）必背
$$\dfrac{x^2}{a^2} + \dfrac{y^2}{b^2} = 1\quad(a>b>0)$$ $$c^2 = a^2 - b^2,\quad e = \dfrac{c}{a}\in(0,1)$$

※ 椭圆是 $a^2 - b^2$；双曲线才是 $a^2 + b^2$，别混了。

抛物线 · 标准方程必背
$$y^2 = 2px\;(p>0)$$

焦点 $(\tfrac{p}{2}, 0)$，准线 $x = -\tfrac{p}{2}$。
焦半径：$|PF| = x_0 + \tfrac{p}{2}$（横坐标加半个 p）。

弦长公式救命
$$|AB| = \sqrt{1+k^2}\cdot|x_1 - x_2| = \sqrt{1+k^2}\cdot\sqrt{(x_1+x_2)^2 - 4x_1 x_2}$$

把 $x_1+x_2$、$x_1 x_2$ 代入韦达定理结果即可，不用解出 $x_1$、$x_2$。

点到直线距离高频
$$d = \dfrac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}}$$

直线写成 $Ax + By + C = 0$ 标准形式，注意分子加绝对值。

双曲线 · 标准方程（焦点在 x 轴）必背
$$\dfrac{x^2}{a^2}-\dfrac{y^2}{b^2}=1\quad(a>0,b>0)$$ $$c^2=a^2+b^2,\quad e=\dfrac{c}{a}>1$$

※ 双曲线是 $a^2+b^2=c^2$；椭圆是 $a^2-b^2=c^2$，别混！

椭圆焦点弦 / 通径技巧
通径（过焦点垂直于长轴的弦）长 $=\dfrac{2b^2}{a}$。
焦半径：$|PF_1|=a+ex_0$，$|PF_2|=a-ex_0$。

2真题示范

2023 · 新高考I卷 · 第21题（节选）
已知椭圆 $C:\dfrac{x^2}{a^2}+\dfrac{y^2}{b^2}=1(a>b>0)$ 的离心率为 $\dfrac{\sqrt{5}}{3}$，点 $A(-2,0)$ 在 $C$ 上。
(1) 求 $C$ 的方程；
(2) 过点 $(-2,3)$ 的直线交 $C$ 于 $P,Q$ 两点，直线 $AP,AQ$ 与 $y$ 轴交于 $M,N$ 两点，证明：线段 $MN$ 的中点为定点。
第（1）问 · 送分
  1. $A(-2,0)$ 在椭圆上 $\Rightarrow a=2$
  2. $e=\dfrac{c}{a}=\dfrac{\sqrt{5}}{3}$ $\Rightarrow c=\dfrac{2\sqrt{5}}{3}$
  3. $b^2=a^2-c^2=4-\dfrac{20}{9}=\dfrac{16}{9}$，所以方程为 $\dfrac{x^2}{4}+\dfrac{9y^2}{16}=1$
第（2）问 · 套路：设直线 → 联立 → 韦达 → 代入
  1. 设直线 $PQ:y-3=k(x+2)$，即 $y=kx+2k+3$
  2. 联立椭圆方程：$\dfrac{x^2}{4}+\dfrac{9(kx+2k+3)^2}{16}=1$
  3. 整理得 $(4+9k^2)x^2+18k(2k+3)x+9(2k+3)^2-16=0$
  4. 设 $P(x_1,y_1),Q(x_2,y_2)$，由韦达：$x_1+x_2=-\dfrac{18k(2k+3)}{4+9k^2}$，$x_1x_2=\dfrac{9(2k+3)^2-16}{4+9k^2}$
  5. 直线 $AP:y=\dfrac{y_1}{x_1+2}(x+2)$，令 $x=0$ 得 $M\left(0,\dfrac{2y_1}{x_1+2}\right)$；同理 $N\left(0,\dfrac{2y_2}{x_2+2}\right)$
  6. 中点纵坐标 $=\dfrac{1}{2}\left(\dfrac{2y_1}{x_1+2}+\dfrac{2y_2}{x_2+2}\right)=\dfrac{y_1}{x_1+2}+\dfrac{y_2}{x_2+2}$，代入 $y=kx+2k+3$ 化简得 $=3$
  7. 所以 $MN$ 中点为定点 $(0,3)$

※ 第（2）问即使没证出中点是定点，设直线、联立、写韦达、写 $M,N$ 坐标这四步写出来就有 6 分。圆锥曲线是"步骤分之王"。

3这些坑别踩

  • 直线没考虑斜率不存在：设 $y = kx + b$ 漏了 $x = m$ 这种竖直线情况
  • $\Delta > 0$ 没写：联立得二次方程要写判别式大于零（保证两交点），常考踩分点
  • 焦点弦公式记错：抛物线 $|PF| = x_0 + \tfrac{p}{2}$，椭圆是 $a - ex_0$ / $a + ex_0$
  • 联立后忘了用韦达：直接硬解 $x_1$、$x_2$ 出根号，运算量爆炸
