1公式速查

公式过长时可左右滑动 ↔

椭圆 · 标准方程（焦点在 x 轴）必背
$$\dfrac{x^2}{a^2} + \dfrac{y^2}{b^2} = 1\quad(a>b>0)$$ $$c^2 = a^2 - b^2,\quad e = \dfrac{c}{a}\in(0,1)$$

※ 椭圆是 $a^2 - b^2$；双曲线才是 $a^2 + b^2$，别混了。

抛物线 · 标准方程必背
$$y^2 = 2px\;(p>0)$$

焦点 $(\tfrac{p}{2}, 0)$，准线 $x = -\tfrac{p}{2}$。
焦半径：$|PF| = x_0 + \tfrac{p}{2}$（横坐标加半个 p）。

弦长公式救命
$$|AB| = \sqrt{1+k^2}\cdot|x_1 - x_2| = \sqrt{1+k^2}\cdot\sqrt{(x_1+x_2)^2 - 4x_1 x_2}$$

把 $x_1+x_2$、$x_1 x_2$ 代入韦达定理结果即可，不用解出 $x_1$、$x_2$。

点到直线距离高频
$$d = \dfrac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}}$$

直线写成 $Ax + By + C = 0$ 标准形式，注意分子加绝对值。

3这些坑别踩

  • 直线没考虑斜率不存在：设 $y = kx + b$ 漏了 $x = m$ 这种竖直线情况
  • $\Delta > 0$ 没写：联立得二次方程要写判别式大于零（保证两交点），常考踩分点
  • 焦点弦公式记错：抛物线 $|PF| = x_0 + \tfrac{p}{2}$，椭圆是 $a - ex_0$ / $a + ex_0$
  • 联立后忘了用韦达：直接硬解 $x_1$、$x_2$ 出根号，运算量爆炸
