选填技巧 · 不会做也能蒙对

高考数学选填占 80 分，做对 60% 就有 48 分托底。这 5 个方法专治"会做但来不及"和"不会做但又不能空"。

先看这个：选填的真正敌人是时间

高考数学选择 8 道、填空 3-4 道，建议时间 40 分钟以内。
意思是平均每道 3-4 分钟，一道做不出立刻用技巧蒙——磨到 8 分钟，后面大题就崩了。

※ 选填策略口诀：能算就算，算不出就蒙，蒙不准就跳过，最后再回头。

1特殊值法

题目说"对任意 $x\in\mathbb{R}$"、"对所有满足条件的…"——立即代特殊值进去验证。

最常用的特殊值
$$x = 0,\;1,\;-1,\;\dfrac{1}{2},\;\dfrac{\pi}{4},\;\dfrac{\pi}{2}$$

三角题用 $\tfrac{\pi}{4}$ 或 $\tfrac{\pi}{2}$，函数题用 $0,1,-1$，含 $a^2$ 的用 $a=1$ 或 $a=2$。

用法示范
若函数 $f(x) = ax^3 + bx + c$ 满足 $f(-x) = -f(x)$，则下列哪项一定成立？
A. $a = 0$　B. $b = 0$　C. $c = 0$　D. $a + b = 0$
直接代 $x = 0$： $f(0) = c$，$-f(0) = -c$，由 $f(-0) = -f(0)$ 得 $c = -c$，所以 $c = 0$。
3 秒选 C。

2选项互斥/包含排除

看选项之间的逻辑关系——如果两个选项互相矛盾，答案大概率在它俩之中；如果一个包含另一个，被包含的几乎不可能是答案。

三种典型情况
  • 互斥型：A 说"$a > 0$"，B 说"$a < 0$"——答案多半是 A 或 B
  • 包含型：A 说"$x \in (1,3)$"，B 说"$x \in (1,5)$"——A 是 B 的子集，B 范围更全更可能对
  • 对称型：4 个选项有 3 个长得很像、1 个特别——往往那 3 个像的里有答案

※ 高考出题人不会出"明显错"的选项，每个选项都是用错误思路推出来的——你看选项就能反推可能的解题方向。

3量纲 / 奇偶 / 边界值检验

算完一个答案不放心？用这三招快速验：

量纲检验

体积题答案带 $\pi$ 没？面积题单位对吗？概率值 必须在 $[0,1]$，方差必须 $\ge 0$。

奇偶性检验

题目说"$f(x)$ 为奇函数"，最后答案应该满足 $f(-x) = -f(x)$；偶函数则 $f(-x) = f(x)$。代一个值反向验证。

边界值检验

求范围题（如 $a \in (?,?)$），代端点值验证：端点要不要取？等号要不要带？这是高考最常踩的分。

4填空题端点效应（"猜整数"）

填空题答案以 整数、简单分数、$\sqrt{2},\sqrt{3},\pi$ 之类为主——奇形怪状的答案大概率算错了。

高考填空答案的"长相"
  • 整数：$0, 1, 2, 3, -1, -2$ 居多
  • 简单分数：$\tfrac{1}{2}, \tfrac{1}{3}, \tfrac{3}{4}$
  • 含根号：$\sqrt{2}, \sqrt{3}, 2\sqrt{2}, \tfrac{\sqrt{3}}{2}$
  • 含 $\pi$：$\tfrac{\pi}{6}, \tfrac{\pi}{4}, \tfrac{\pi}{3}, \tfrac{2\pi}{3}$
  • 含 $e$ 或 $\ln$：导数题常见

※ 算出 $\dfrac{17}{23}$ 这种诡异数字——99% 是某一步算错了。回头查上一步。

5多选题策略：宁缺毋滥

新高考多选题的得分规则：全对 6 分 / 部分对 3 分 / 选错 0 分——所以"少选"比"多选"安全得多。

三档应对策略
  • 100% 确定的选项：选上
  • 50% 把握的选项：不选（多选错一个就 0 分，宁可少拿 3 分也别赌）
  • 完全不会判断的：不选
取舍示范
某多选题选项 ABCD，你判断：A 一定对、B 大概率对（70%）、C 不知道、D 大概率错。
正确策略：只选 A，拿 3 分。
为什么不选 B？因为如果 B 错，整题 0 分；只选 A 至少保 3 分。3 分到手好过 6 分赌赢 70%、赌输 30%。

※ 数学期望算给你看：
只选 A：稳拿 3 分
选 AB：$0.7 \times 6 + 0.3 \times 0 = 4.2$ 分（但风险高）
看起来期望高，但 30% 拿 0 分的体验在考场上会把你击溃。

最后一句话

技巧不能替代会做，但能替你抢回时间。
考场上前 40 分钟选填能稳住 60+ 分，
大题写不完也已经赢了一半。
